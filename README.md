## ⚙️ 遇到的問題與解決方式 | 問題と解決方法
| 問題 | 中文解釋 | 日本語説明 |
|------|------------|------------|
| JavaFX 視窗未顯示 | 不熟悉 Application 結構，後來了解需覆寫 `start(Stage)`。 | `Application` の構造を理解しておらず、`start(Stage)` を正しく実装して解決。 |
| Maven 無法編譯 | `pom.xml` 未設定 javafx 外掛。 | `pom.xml` に `javafx-maven-plugin` を追加して解決。 |
| module not found | 不懂 `module-info.java` 的用途。 | `module-info.java` の `requires` を正しく設定して解決。 |
| jpackage 打包失敗 | 未安裝 WiX Toolset 或 PATH 錯誤。 | WiX 3.14 をインストールし、環境変数を修正。 |
| 刪除 target 錯誤 | `.exe` 被執行中鎖定無法刪除。 | `.exe` が実行中で削除できなかったため、`taskkill` で終了して再ビルド。 |
